BaseClass

Encapsulates the common methods of the amoRF toolbox. The methods are reserved for later use by the graphical user interface.

Properties

  • Name - public

Methods

  • write
  • read
  • getstructure

See also: SignalClass, AttributeClass, BlockClass, TunerClass.

Name

Name of the the object. Default values is empty string.

write@BaseClass

[OBJECT] = write(OBJECT,'PROPERTY',VALUE)

The method saves VALUE (numeric) to PROPERTY (string) field of the OBJECT. The updated object is returned in OBJ.

read@BaseClass

[VALUE] = write(OBJECT,'PROPERTY')

The method returns the VALUE of OBJECTS's PROPERTY.

getstructure@BaseClass

ROOT = getstructure(OBJECT)

The method returns the object properties in a uitree structure.
