FreqSelectClass

FreqSelectClass is used to implement filtering and frequency dependent nonidealities

Properties

  • flt

Methods

  • process

flt

This member is a concatenation of a filter nominator and denominator. Best recommended to be used with MATLAB filter design functions.

process

output=obj.process(signal)

The input signal is recommended to be an object of SignalClass. The return value output is also an object of SignalClass. The process function will apply the filter given by flt on signal.
